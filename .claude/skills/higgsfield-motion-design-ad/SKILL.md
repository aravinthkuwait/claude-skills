# Seedance 2.0 on Higgsfield — Motion Design Ad Generator

## Overview
This skill creates motion design advertisement prompts for software, SaaS, and tech companies using Seedance 2.0 on Higgsfield. Covers app demos, feature launches, startup promos, and UI showcases.

## Prompt Structure
[Product type] [Key message/USP] [Visual style] [Animation style] [Color/branding] [CTA moment] [Duration/pacing]

## Ad Style Keywords
- **SaaS/App**: "clean UI animation", "dashboard reveal", "feature highlight", "onboarding flow"
- **Startup**: "bold brand reveal", "mission statement animation", "founder story", "growth metrics"
- **Product Launch**: "countdown reveal", "feature showcase", "before/after", "problem/solution"
- **Corporate Tech**: "data visualization", "network animation", "AI/ML visualization", "cloud infrastructure"

## Motion Design Techniques
- **Kinetic Typography**: Animated text with purpose and rhythm
- **UI Animation**: Screen recordings with smooth transitions and highlights
- **Isometric Design**: 3D isometric illustrations in motion
- **Data Viz**: Animated charts, graphs, and infographics
- **Particle Systems**: Abstract tech particles forming shapes/logos

## Color & Branding
Gradient overlays, brand color palettes, dark mode aesthetics, glassmorphism effects, neon accents on dark backgrounds

## Timing & Pacing
- 15-second social cut (hook → problem → solution → CTA)
- 30-second standard ad (full story arc)
- 60-second brand film (deep narrative)
